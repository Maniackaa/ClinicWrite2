# Быстрая установка бота на Ubuntu 24.04

## Для Ubuntu 24.04 (Noble) - используйте эти команды:

```bash
# 1. Обновление системы и установка пакетов
sudo apt update
sudo apt install -y python3 python3-venv python3-pip git curl libpq-dev postgresql-client build-essential

# 2. Клонирование репозитория
cd /opt
sudo git clone https://github.com/Maniackaa/ClinicWrite.git clinic-bot
cd clinic-bot

# 3. Развертывание
sudo chmod +x deploy.sh
sudo ./deploy.sh

# 4. Настройка .env файла
sudo nano /opt/clinic-bot/.env
# Заполните: BOT_TOKEN, ADMIN_IDS, GROUP_ID

# 5. Запуск
sudo systemctl start clinic-bot
sudo systemctl enable clinic-bot
sudo systemctl status clinic-bot
```

## Проверка версии Python

```bash
python3 --version
```

В Ubuntu 24.04 должно быть: `Python 3.12.x`

Скрипт `deploy.sh` автоматически определит доступную версию Python и использует её.
