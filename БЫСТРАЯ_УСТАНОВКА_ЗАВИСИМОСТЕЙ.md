# Быстрая установка зависимостей на сервере

## Команды для выполнения:

```bash
cd /opt/clinic-bot

# 1. Установите системные зависимости
sudo apt-get install -y libpq-dev postgresql-client build-essential python3-dev pkg-config

# 2. Установите зависимости Python (игнорируя ошибки с системными пакетами)
sudo python3 -m pip install -r requirements.txt --break-system-packages --no-warn-script-location

# 3. Проверьте установку
python3 -c "import structlog; import aiogram; print('OK')"

# 4. Проверьте логи бота для диагностики
sudo journalctl -u clinic-bot -n 100
```

## Если возникают ошибки с системными пакетами:

```bash
# Установите только недостающие пакеты вручную
sudo python3 -m pip install structlog aiogram apscheduler environs pytz --break-system-packages --no-warn-script-location
```

## Проверка работы бота:

```bash
# Запуск
sudo systemctl start clinic-bot

# Статус
sudo systemctl status clinic-bot

# Логи
sudo journalctl -u clinic-bot -f
```
